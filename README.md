# 🐉 Dragons Hub - Omni X Definitive Script

Bem-vindo ao repositório oficial do **Dragons Hub**! Este é o script definitivo (Hub) desenvolvido para maximizar a sua experiência no jogo **Omni X** (Roblox). Nosso objetivo é fornecer ferramentas automatizadas, seguras e práticas para facilitar o seu progresso no jogo.

## 🚀 Funcionalidades Principais

O Dragons Hub conta com diversas funções essenciais para a sua gameplay:

- **🗡️ Auto Raid Aprimorado:** Entre e complete Raids automaticamente. O script interage direto com os NPCs, otimizando o seu tempo de farm.
- **📈 Auto Farm & XP Alternativo:** Ganhe experiência rapidamente completando missões em loop (inclui rotas exclusivas, como assalto ao banco para a classe de Vilão).
- **🗺️ Sistema de Teleportes:** Vá instantaneamente para pontos de interesse cruciais do mapa, incluindo Raids, seletor de Classes (ex: Osmosiano), Times e NPCs importantes como o Azmuth.
- **✨ Interface de Usuário (UI) Moderna:** Um menu intuitivo e prático, projetado para ser limpo e fácil de navegar durante o jogo.

## 🛠️ Como Utilizar

Para executar o Dragons Hub, você precisará de um executor de scripts compatível com o Roblox (como Delta, Fluxus, Krnl, etc.).

1. Abra o seu executor de scripts preferido.
2. Inicie o Roblox e entre no jogo **Omni X**.
3. Copie o código de execução (Loadstring) abaixo.
4. Cole no seu executor e clique no botão de **Executar (Execute)**.
5. A interface do **Dragons Hub** aparecerá na sua tela. Divirta-se!

### 💻 Loadstring (Copie e Cole)

```lua
-- Execute o Dragons Hub no Omni X
loadstring(game:HttpGet("https://raw.githubusercontent.com/Dragons-Forge/dragons-hub/main/omnix-definitive.lua"))()
```
